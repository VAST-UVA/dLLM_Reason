### V1.5.0 — Adaptive Dynamic DAG + LaTeX Tables

| Component | Details |
|-----------|---------|
| `scheduler/adaptive_dynamic_scheduler.py` | **Core novel contribution**: Dynamic soft DAG that constructs pairwise influence graph at runtime. Computes cosine similarity of logit distributions, builds soft dependency graph, selects positions by readiness x confidence. |
| `scripts/generate_latex_table.py` | Publication-ready booktabs LaTeX tables from summary.json. Auto-detects strategies/benchmarks, bolds best-per-column. |
| `scripts/runs/full_comparison.sh` | One-command full comparison: 13 strategies x 10 benchmarks |

### V1.4.0 — SpanDAG + DAGScheduler Dimension Fix

| Component | Details |
|-----------|---------|
| `graph/span_dag.py` | Coarse-grained DAG over token spans — reduces search space by span_size^2. Factories: empty, linear_chain, cot, from_levels, from_token_dag. Mutation operators for search. |
| `scheduler/dag_scheduler.py` | **Critical fix**: DAG adjacency is (gen_len, gen_len) but sampler passes (prompt_len + gen_len) tensors. Added auto-detection of prompt_len, slice generation area for ready_positions(), pad back to full sequence. |
